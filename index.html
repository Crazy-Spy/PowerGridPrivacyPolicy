Política de Privacidade - Power Grid

O aplicativo Power Grid foi desenvolvido como um app Gratuito. Este serviço é fornecido sem custo e destina-se ao uso como está.

1. Coleta de Informações

Nós não coletamos, solicitamos ou armazenamos quaisquer dados pessoais identificáveis diretamente através do aplicativo.

2. Serviços de Terceiros

O app utiliza serviços de terceiros que podem coletar informações usadas para identificar o seu dispositivo para fins de funcionamento técnico:

  • Google Play Services

3. Rankings e Desafios (Futuro)

Para o sistema de ranking global e desafios diários, poderemos utilizar identificadores anônimos ou apelidos escolhidos pelo usuário para exibir pontuações. Esses dados são usados exclusivamente para fins competitivos dentro do jogo.

4. Segurança

Valorizamos sua confiança. Como não coletamos dados pessoais, o risco de exposição de informações sensíveis é inexistente.

5. Contato

Se você tiver dúvidas sobre esta política, entre em contato pelo e-mail: flavio@example.com
